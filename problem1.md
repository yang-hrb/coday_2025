# Effective Taverns of NiCE Realm

## 背景描述

在广阔的 NiCE 王国中，城市如珍珠般散落在漫长的道路上。旅行者们渴望在舒适的酒馆中享用“能量与清醒的魔法药剂”。

为了确保公平，行会决定制定一个新的标准：**找到任何旅行者从他们的城市到最近的酒馆必须步行的最长路径**。这个数值将作为新的最大距离标准，承诺任何人都无需为了休息而走得比这更远。

Quantus（计算大师）发现距离仅仅是差值。他计算了每个城市到其最近酒馆的距离，并在这些距离中找到了一个最大值。这个单一的数值代表了王国中最艰难的旅程，即需求最大的地方。

## 题目要求

给定一系列城市的位置和一系列酒馆的位置（均位于一条直线上），请计算出所有城市中，“该城市到最近酒馆的距离”的最大值。

换句话说：

1. 对于每一个城市 $C_i$，找到离它最近的酒馆 $T_j$，计算距离 $d_i = |C_i - T_j|$。
2. 在所有城市的 $d_i$ 中，找到最大值 $\max(d_1, d_2, \dots, d_n)$。

## 输入格式 (Input Format)

输入包含 4 行：

1. 第一行：一个整数 $N$，表示城市的数量。
2. 第二行：$N$ 个整数，表示每个城市在直线地图上的位置坐标。
3. 第三行：一个整数 $M$，表示酒馆的数量。
4. 第四行：$M$ 个整数，表示每个酒馆在直线地图上的位置坐标。

## 输出格式 (Output Format)

* 输出一个整数，表示任何城市到其最近酒馆的最大距离。


## 数据范围 (Constraints)

* **城市数量 ($N$)**: $1 \le N \le 10^5$ (根据 PDF 示例推断)
* **酒馆数量 ($M$)**: $1 \le M \le 10^5$ (根据 PDF 示例推断)
* **位置坐标**: $-10^9 \le \text{Location} \le 10^9$ (根据 PDF 示例推断，位置为整数)
